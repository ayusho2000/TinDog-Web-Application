# 🐶 TinDog – Dog Dating Website

TinDog is a responsive landing page website designed to help dogs meet new and interesting friends nearby. The project is inspired by modern dating platforms and provides a fun interface where dog owners can find potential matches for their pets.

This project focuses on building a clean and responsive UI using Bootstrap components and modern web design principles.

---

##  Features

- Attractive and modern landing page design
- Fully responsive layout
- Clean UI built using Bootstrap
- Pricing plans for different dog categories
- Testimonials from happy dog users
- Download options for mobile applications
- Easy navigation and user-friendly interface

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Bootstrap**
- **SVG Icons**
- **Responsive Web Design**
